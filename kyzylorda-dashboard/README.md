## Kyzylorda City Incident & Infrastructure Map

A modern web dashboard built with **React**, **Tailwind CSS**, **Lucide React icons**, and **React Leaflet** to visualize incidents and infrastructure events across **Kyzylorda** on an interactive map.

### Features

- **Full-screen interactive map** centered on Kyzylorda (44.8488, 65.4823) using `react-leaflet`.
- **Glassmorphism sidebar** on the left showing a live-style news feed of events.
- **Global search bar** at the top with **street name suggestions** that recenter the map.
- **Dark theme** with **Electric Blue** and **Warning Orange** accent colors.
- **Mock events JSON data** for repair, emergency, and road work incident types.

### Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open the printed local URL in your browser.

### Tech Stack

- React + Vite
- Tailwind CSS
- Lucide React icons
- React Leaflet + Leaflet
